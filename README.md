# Electrify ⚡

## 🧾 Overview

Electrify is a full-stack e-commerce web application built with Flask that enables users to browse products, manage a shopping cart, and complete secure purchases. The project demonstrates real-world backend development concepts including authentication, database design, and RESTful architecture.

The goal of this project is to simulate a production-level online shopping experience with clean architecture and scalable design.

---

## ✨ Features

* User registration and authentication system
* Product listing and detailed product pages
* Shopping cart functionality
* Add/remove items from cart
* Order processing workflow
* Admin product management (if applicable)
* Responsive UI for mobile and desktop users

---

## 🛠️ Tech Stack

* **Backend:** Flask (Python)
* **Database:** PostgreSQL / SQLite (depending on deployment)
* **ORM:** SQLAlchemy
* **Frontend:** HTML, CSS, JavaScript and Bootstrap
* **Authentication:** Flask-Login 
* **Version Control:** Git & GitHub

---

## 🧠 Key Concepts Demonstrated

* Full-stack web development using Flask
* RESTful routing and API structure
* Database modeling and relationships
* User authentication and session management
* CRUD operations
* Deployment of web applications

## ⚙️ Installation & Setup

```bash
git clone https://github.com/your-username/electrify.git
cd electrify

python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate

pip install -r requirements.txt

python app.py
```
